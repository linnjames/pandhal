TCS/TDS

This module helps to calculate TCS/TDS in invoicing.



Invoicing

Step 1:

In Invoicing -> Configuration -> Setting , TCS/TDS should be selected.








Step 2:

In Invoicing -> Configuration -> TDS or TCS, Add New TDS / TCS








TCS Master







TDS Master







Step 3:

In Invoicing -> Sales -> Customers -> Sales & Purchases, TCS Applicable should be selected.








In Invoicing -> Purchases -> Vendors -> Sales & Purchases, TDS Applicable should be selected.








Step 5:

In Invoicing -> Sales -> Customer Invoices. Create Customer Invoice, If TCS Applicable for Selected Customer then TCS Calculate in Invoice.













Step 5:

In Invoicing -> Purchases -> Vendor Bills. Create Vendor Bill, If TDS Applicable for Selected Customer then TDS Calculate in Invoice.
