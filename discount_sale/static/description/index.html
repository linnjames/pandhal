Discount Sales

Overview of Discount Sales

Discount Sale is mainly focused on the purpose of the Approval from the discount sales. It will help to sort the sale orders that above a particular percentage of discount.

Configuration

1. Go to sales -> Configuration -> Settings -> On Sales Head Enable Discount -> Save
